# Hello

Welcome to my portfolio website! This is a personal showcase of my work, skills, and projects.

## Website Sections

- **Hello Animation** - Animated greeting with fireworks effect
- **Who am I?** - Personal summary and technologies I work with
- **My Projects** - Portfolio of applications and projects I've built
- **Send me a message** - Contact form and social media links

## Technology Stack

- **React** 18.3.1 - UI library
- **TypeScript** 5.4.5 - Type-safe JavaScript
- **Vite** 5.2.0 - Build tool and development server
- **React Router DOM** 6.26.0 - Client-side routing
- **React Icons** 5.5.0 - Icon library
- **Sass** 1.77.0 - CSS preprocessor
- **Docker** - Containerization
- **Nginx** - Web server for production
